1.find 命令 （格式： find [-path] -options
path        表示要在哪个路径下查找 ，默认情况下在当前目录下
-name       按文件名的某种规则进行查找
文件明的规则（* 为通配符 匹配任意内容、 ？ 为 任意匹配一个字符、 [] 匹配任意一个中括号内的字符
例如： find -name '*.pyc'

-size       按文件的大小查找
-type       按文件的类型（f 为普通文件、l 为符号连接、d 为目录）


2.xargs 命令（与find命令结合使用，用管道符 '|' 链接 , 可以把第一个指令 得到结果作为参数 传给xargs命令，然后进行执行）
find ………… | xargs ……    （执行多命令）
例如： find ./ -name '*.py' | xargs cat (查看搜索到的文件的内容）

可以使用 sudo Python3 ***.py 来执行Python文件（必须提权）


3.grep命令 （查找并 显示文件的内容） 格式： grep -param file
-c      计算符合样式的列数
-i      忽略字符大小写的差别
-n      显示时标出序号
-s      不显示错误信息  (一般报错为 '是以一个文件夹' 使用 -s 后就不显示了 ) 
-v      反转查找

注意:
管道符 与 xargs 的区别 
管道符 传出参数 是字符串类型
xargs 是将 上一条命令的结果 作为 参数传给下一个命令，不是字符串

列如:下面两条命令
find  -name '1.sh' | xargs cat
find  -name '1.sh' | cat
第一条 可查看 1.sh 文件的内容，第二条 只能 打印文件的名字
